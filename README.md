## Install

[npm][]:

```sh
npm install @breejs/ts-worker
```

[yarn][]:

```sh
yarn add @breejs/ts-worker
```


## Usage

```js
const @breejs/tsWorker = require('@breejs/ts-worker');

const @breejs/tsWorker = new @breejs/tsWorker();

console.log(@breejs/tsWorker.renderName());
// script
```
